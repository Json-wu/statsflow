# Browser History Stats 📊

一个简洁实用的 Chrome 扩展，用于统计浏览器历史记录。

## 功能特性

- ✅ **合并相同站点** - 自动按域名合并，避免重复
- ✅ **访问次数统计** - 清晰展示每个站点的访问频率
- ✅ **页面数统计** - 显示每个站点下访问的不同页面数量
- ✅ **首次访问时间** - 显示每个站点的首次访问日期
- ✅ **访问占比条** - 直观展示各站点访问量占比
- ✅ **访问时段分布** - 0–23 点访问量柱状图
- ✅ **排序切换** - 支持按最近访问 / 访问次数排序
- ✅ **时间筛选** - 支持全部/今天/本周/本月/日历选日
- ✅ **CSV 导出** - 导出当前筛选结果为 CSV 文件
- ✅ **暗色模式** - 支持亮/暗主题切换
- ✅ **搜索增强** - 支持正则匹配（标题、URL、域名）
- ✅ **访问趋势图** - 按天/按周展示近 7 天或 4 周访问量
- ✅ **隐私控制** - 域名黑名单、按站点删除历史
- ✅ **一键访问** - 点击直接打开站点

## 安装方法

1. 下载或克隆本项目
2. 打开 Chrome，访问 `chrome://extensions/`
3. 开启右上角「开发者模式」
4. 点击「加载已解压的扩展程序」
5. 选择本项目文件夹

## 使用说明

- 点击扩展图标即可查看历史统计
- 点击「全部/今天/本周/本月」按钮筛选时间范围
- 使用排序下拉框切换「最近访问」或「访问次数」排序
- 在搜索框输入关键词搜索，勾选「正则」可使用正则表达式
- 点击 ⚙️ 进入隐私设置，可添加域名黑名单
- 点击站点右侧 🗑 可删除该站点的浏览历史
- 点击任意站点可直接跳转访问

## 技术栈

- Vanilla JavaScript
- Chrome Extensions API (history, storage)
- 纯 CSS 样式
